# 타임박스 집중 관리 서비스 Information Architecture (IA)

## 1. 사이트맵 (사이트맵)

```
홈 (/)
├── 인증 (/auth)
│   ├── 로그인 (/auth/login)
│   ├── 회원가입 (/auth/signup)
│   └── 비밀번호 재설정 (/auth/reset-password)
├── 대시보드 (/dashboard) [인증 필요]
│   ├── 오늘 일정 (/dashboard/today)
│   ├── 주간 보기 (/dashboard/week)
│   └── 월간 보기 (/dashboard/month)
├── 타이머 (/timer) [인증 필요]
│   ├── 집중 모드 (/timer/focus)
│   └── 휴식 모드 (/timer/break)
├── 리포트 (/reports) [인증 필요]
│   ├── 일일 리포트 (/reports/daily)
│   ├── 주간 리포트 (/reports/weekly)
│   └── 월간 리포트 (/reports/monthly)
├── 설정 (/settings) [인증 필요]
│   ├── 프로필 (/settings/profile)
│   ├── 알림 설정 (/settings/notifications)
│   ├── 타이머 설정 (/settings/timer)
│   └── 계정 관리 (/settings/account)
└── 고객 지원 (/support)
    ├── 도움말 (/support/help)
    ├── 튜토리얼 (/support/tutorial)
    └── 문의하기 (/support/contact)
```

## 2. 사용자 흐름 (사용자 흐름)

### **주요 작업: 첫 타임박스 생성 및 실행**
1. 사용자가 홈페이지 방문
2. '시작하기' 버튼 클릭
3. 회원가입 페이지로 이동
4. 이메일, 비밀번호 입력 후 가입
5. 온보딩 튜토리얼 (60초 가이드)
6. 대시보드/오늘 일정 페이지로 자동 이동
7. 드래그 앤 드롭 또는 '+ 타임박스 추가' 버튼 클릭
8. 시간대, 작업명, 카테고리 입력
9. 타임박스 생성 완료
10. 시작 시간 5분 전 알림 수신 (모바일/데스크톱)
11. 타이머 페이지에서 집중 모드 시작
12. 작업 완료 후 '완료' 버튼 클릭

### **주요 작업: 일일 리포트 확인**
1. 대시보드에서 '오늘의 성과' 섹션 확인
2. '자세한 리포트 보기' 클릭
3. 리포트/일일 리포트 페이지로 이동
4. 완료율, 집중 시간, 휴식 패턴 그래프 확인
5. 개선 제안사항 및 다음날 목표 설정

### **주요 작업: 모바일에서 타이머 사용**
1. 모바일 브라우저에서 서비스 접속
2. PWA 설치 안내 (선택사항)
3. 대시보드에서 현재 진행 중인 타임박스 확인
4. '타이머 시작' 버튼 탭
5. 풀스크린 타이머 모드 진입
6. 집중 시간 동안 진행률 실시간 확인
7. 종료 알림 및 휴식 시간 안내
8. 완료 체크 또는 연장 선택

## 3. 네비게이션 구조 (네비게이션 구조)

### **글로벌 네비게이션 바 (GNB) - Topbar**
- **위치**: 화면 상단 고정 (높이 64px)
- **배경색**: #1E2A38 (딥 네이비)
- **구성 요소**:
  - 좌측: 로고 + 서비스명
  - 중앙: 빠른 검색바 (타임박스 검색)
  - 우측: 알림 아이콘 + 사용자 프로필 드롭다운

### **사이드 패널 네비게이션**
- **위치**: 좌측 패널 (너비 280px, 축소 시 64px)
- **구성 요소**:
  - 미니 캘린더 위젯
  - 주요 메뉴 아이콘 + 텍스트
    - 대시보드 (홈 아이콘)
    - 타이머 (시계 아이콘)
    - 리포트 (차트 아이콘)
    - 설정 (톱니바퀴 아이콘)
  - 접기/펼치기 토글 버튼

### **모바일 네비게이션**
- **하단 탭 바**: 주요 4개 메뉴 (대시보드, 타이머, 리포트, 설정)
- **햄버거 메뉴**: 추가 기능 및 고객 지원 메뉴

### **푸터 네비게이션**
- **위치**: 페이지 하단
- **구성 요소**: 개인정보처리방침, 이용약관, 고객센터, 회사 정보

## 5. 콘텐츠 구성 (콘텐츠 구성)

| 페이지 | 주요 콘텐츠 요소 |
|---|---|
| 홈 (/) | 히어로 섹션, 핵심 기능 소개, 사용법 3단계, CTA 버튼, 사용자 후기 |
| 로그인 (/auth/login) | 로그인 폼, 소셜 로그인 버튼, 비밀번호 찾기 링크 |
| 회원가입 (/auth/signup) | 가입 폼, 이용약관 동의, 이메일 인증 안내 |
| 대시보드 - 오늘 (/dashboard/today) | 타임라인 뷰, 타임박스 카드, 진행률 표시, 빠른 추가 버튼 |
| 대시보드 - 주간 (/dashboard/week) | 주간 캘린더 그리드, 일별 완료율, 주간 목표 설정 |
| 타이머 - 집중 (/timer/focus) | 원형 타이머, 현재 작업명, 진행률 바, 일시정지/중단 버튼 |
| 타이머 - 휴식 (/timer/break) | 휴식 타이머, 휴식 활동 제안, 다음 작업 미리보기 |
| 일일 리포트 (/reports/daily) | 완료율 도넛 차트, 시간대별 집중도 그래프, 성과 요약 |
| 주간 리포트 (/reports/weekly) | 일별 완료율 막대 그래프, 주간 트렌드, 개선 제안 |
| 프로필 설정 (/settings/profile) | 사용자 정보 폼, 아바타 업로드, 목표 설정 |
| 알림 설정 (/settings/notifications) | 알림 유형별 ON/OFF 토글, 알림 시간 설정, 테스트 버튼 |

## 6. 인터랙션 패턴 (인터랙션 패턴)

### **데이터 입력 패턴**
- **드래그 앤 드롭**: 타임박스 생성 및 시간 조정
- **자연어 입력**: "오후 2시부터 3시까지 회의" 형태의 빠른 입력
- **인라인 편집**: 타임박스 제목 더블클릭으로 즉시 수정

### **피드백 패턴**
- **토스트 알림**: 성공/오류 메시지 (3-5초 자동 사라짐)
- **진행률 표시**: 실시간 타이머 진행률, 일일/주간 목표 달성률
- **상태 표시**: 타임박스 상태 (예정/진행중/완료/건너뜀)

### **확인 패턴**
- **모달 다이얼로그**: 중요한 작업 삭제 시 확인
- **스와이프 액션**: 모바일에서 완료/삭제 제스처
- **더블 탭**: 빠른 완료 체크

### **도움말 패턴**
- **툴팁**: 아이콘 호버 시 기능 설명
- **온보딩 오버레이**: 첫 방문 시 주요 기능 하이라이트
- **빈 상태 안내**: 타임박스가 없을 때 생성 가이드

## 7. URL 구조 (URL 구조)

### **URL 명명 규칙**
- **일반 리소스**: `/resource-name` (소문자, 하이픈 구분)
- **상세 페이지**: `/resource-name/:id` (동적 파라미터)
- **하위 카테고리**: `/parent/child` (계층 구조 반영)
- **날짜 기반**: `/reports/daily/:date` (YYYY-MM-DD 형식)

### **구체적인 URL 예시**
```
/ - 홈페이지
/auth/login - 로그인
/auth/signup - 회원가입
/dashboard/today - 오늘 대시보드
/dashboard/week/:date - 특정 주 대시보드 (2024-01-15)
/timer/focus/:timeboxId - 특정 타임박스 집중 모드
/reports/weekly/:week - 주간 리포트 (2024-W03)
/settings/notifications - 알림 설정
/support/tutorial - 튜토리얼
```

### **SEO 최적화 고려사항**
- 의미있는 키워드 포함 (timer, focus, productivity)
- 일관된 구조로 크롤링 용이성 확보
- 사이트맵.xml 자동 생성 지원

## 8. 컴포넌트 계층 구조 (컴포넌트 계층 구조)

### **글로벌 컴포넌트**
- **Header**: 로고, 검색바, 사용자 메뉴
- **Sidebar**: 네비게이션 메뉴, 미니 캘린더
- **Footer**: 링크, 저작권 정보
- **Toast**: 알림 메시지 표시
- **Modal**: 확인 다이얼로그, 설정 팝업

### **레이아웃 컴포넌트**
- **AppLayout**: 전체 앱 레이아웃 (Header + Sidebar + Main)
- **AuthLayout**: 인증 페이지 전용 레이아웃
- **TimerLayout**: 풀스크린 타이머 레이아웃
- **ReportLayout**: 리포트 페이지 레이아웃

### **비즈니스 컴포넌트**
- **TimeboxCard**: 개별 타임박스 표시 카드
- **TimerWidget**: 원형 타이머 위젯
- **ProgressChart**: 진행률 차트 (도넛, 막대)
- **CalendarGrid**: 주간/월간 캘린더 그리드
- **QuickAddForm**: 빠른 타임박스 추가 폼

### **폼 컴포넌트**
- **TimeboxForm**: 타임박스 생성/편집 폼
- **ProfileForm**: 사용자 프로필 편집 폼
- **SettingsForm**: 각종 설정 폼
- **LoginForm**: 로그인 폼
- **SignupForm**: 회원가입 폼

### **UI 기본 컴포넌트**
- **Button**: Primary, Secondary, Ghost 버튼
- **Input**: 텍스트, 이메일, 비밀번호 입력
- **Select**: 드롭다운 선택
- **Toggle**: ON/OFF 스위치
- **DatePicker**: 날짜 선택기
- **TimePicker**: 시간 선택기

### **모바일 특화 컴포넌트**
- **BottomTabBar**: 하단 탭 네비게이션
- **SwipeAction**: 스와이프 액션 래퍼
- **PullToRefresh**: 당겨서 새로고침
- **FloatingActionButton**: 플로팅 액션 버튼

### **컴포넌트 재사용 우선순위**
1. **높은 재사용성**: Button, Input, Modal, Toast
2. **중간 재사용성**: TimeboxCard, ProgressChart, CalendarGrid
3. **낮은 재사용성**: TimerWidget, 페이지별 전용 폼

### **상태 관리 패턴**
- **전역 상태**: 사용자 인증, 타임박스 데이터, 설정
- **로컬 상태**: 폼 입력, UI 상태 (모달 열림/닫힘)
- **서버 상태**: API 데이터 캐싱 및 동기화

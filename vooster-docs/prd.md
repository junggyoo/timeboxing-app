# 타임박스 집중 관리 웹·모바일 서비스 PRD

## 1. 서비스 개요
시간을 블록 단위로 계획·실행·검토하게 해 주는 가벼운 웹 기반 타임박싱 도구. 데스크톱에서 계획을 세우고, 모바일에서 타이머·알람을 받아 흐름을 지키도록 지원한다.

## 2. 문제 정의
1. 사용자는 타임박스를 만들어도 알람·리마인드 부족으로 계획을 지키지 못한다.
2. 집중·휴식 주기를 놓쳐 과로하거나 금세 지친다.
3. 실천 데이터를 쉽게 확인할 수 없어 동기 부여가 약하다.

## 3. 사용자 목표
계획한 타임박스를 90% 이상 지켜 업무·학습 효율을 높이고, 집중/휴식 사이클을 건강하게 유지한다.

## 4. 대상 사용자
• 집중력을 높이고 싶은 직장인
• 학습 시간을 관리하려는 대학생
• 프로젝트 단위로 일하는 프리랜서

## 5. 핵심 가치 제안
“60초 만에 만든 타임박스를 모바일 알람으로 지키고, 하루·주간 리포트로 성과를 시각화합니다.”

## 6. 주요 기능 (MVP)
1. 빠른 타임박스 일정 생성
   - 드래그 앤 드롭 또는 자연어 입력
2. 집중 타이머 / 카운트다운
   - 현재 타임박스 실시간 진행률 표시
3. 알람 & 휴식 리마인더
   - 시작 5분 전, 종료 알림, 휴식 시간 알림(1시간 집중 기준 5~10분 휴식)
4. 데일리·위클리 사용 리포트
   - 계획 대비 완료 비율, 총 집중 시간, 집중 패턴 그래프
5. 멀티 디바이스 지원
   - 데스크톱·모바일 브라우저 완전 대응, 데이터 실시간 동기화

## 7. 비기능 요구사항
• 반응형 웹(Desktop ≥1280px, Mobile ≥360px)
• 페이지 로딩 3초 이내
• 알람: PWA Push 및 브라우저 Notification API
• 데이터 보관: Supabase PostgreSQL, Region: ap-northeast-1
• 개인정보 및 GDPR/PIPA 준수

## 8. 성공 지표 (MVP 기준)
• Activation: 가입자의 60%가 첫 주 내 타임박스 3개 이상 생성
• Adherence: 평균 55% 이상 타임박스 완료율
• Retention: 주간 재방문율 40% 이상
• NPS: 30 이상

## 9. 출시 범위 & 로드맵
| 단계 | 기간 | 범위 |
|------|------|------|
|M0-M3|MVP 개발·내부 테스트|기능 1~5, 기본 이메일 로그인|
|M3-M4|Closed Beta 50명|버그 수정, UX 개선|
|M4-M6|Open Beta 500명|주간 리포트 이메일, PWA 설치 옵션|
|M6|정식 출시|과금 모델 도입(프리미엄 구독) |

## 10. 수익 모델
프리미엄 구독 ₩5,900/월
• 무제한 타임박스(무료 20개/주)
• 고급 리포트(월간 트렌드·PDF 내보내기)
• 캘린더 연동(Google/Outlook)

## 11. 경쟁 우위
• 60초 온보딩과 웹·모바일 동시 사용으로 진입장벽 최소화
• 집중/휴식 자동 알람으로 ‘실행’에 집중한 구조
• 경쟁 대비 40~70% 저렴한 가격

## 12. 리스크 & 완화 전략
| 리스크 | 완화 |
|---------|------|
|초기 정착률 저조|1·3·7일차 알림, 주간 리포트로 습관 형성|
|캘린더 연동 오류|MVP 단계에서는 연동 제외, 정식 출시 전에 충분한 베타 테스트|
|브라우저 알림 제한|PWA 설치 유도, 이메일/모바일 푸시 병행|

## 13. 개발 전략
Agile MVP → 사용자 피드백 반영 2주 스프린트. 핵심 지표 달성 시에만 기능 확대.
